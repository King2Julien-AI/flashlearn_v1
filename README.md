# FlashLearn — Offline Vocab Trainer (Tauri v2 + Vite)

Clean project export (no build artifacts).

## Structure
- `web/` — frontend (Vite)
- `src-tauri/` — desktop shell (Tauri v2 / Rust)

## Commands

### 1) Install dependencies
```bash
npm install
```

### 2) Run in browser (web dev)
```bash
npm run dev
```

### 3) Run as desktop app (Tauri dev)
```bash
npm run tauri:dev
```

### 4) Build desktop bundles/installers
```bash
npm run tauri:build
```
